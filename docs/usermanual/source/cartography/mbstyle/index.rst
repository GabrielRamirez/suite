Styling with MBStyle
--------------------

MBStyle is a styling language based on the `Mapbox Style Specification <https://www.mapbox.com/mapbox-gl-js/style-spec>`_. It is intended primarily for client-side styling with vector data.

This section will discuss using MBStyle in Boundless Server.
